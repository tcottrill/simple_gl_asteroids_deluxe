# simple_gl_asteroids_deluxe
A very simple Asteroids Deluxe Emulator

Some code in this is copyright the MAME team. 

You will need the following roms from astdelux.zip in the roms directory:

- 036430-02.d1
- 036431-02.ef1
- 036432-02.fh1
- 036433-03.j1
- 036800-02.r2
- 036799-01.np2
- 034602-01.c8
